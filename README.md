# Pokemon TCG Collection Tracker

A web app for tracking Pokémon Trading Card Game collections, viewing card data, and monitoring price changes.

## Features

- Browse Pokémon TCG cards and sets
- Track cards in your personal collection
- View recent price changes
- Discover newly released sets

## Tech Stack

- Nuxt 3
- Vue 3
- Nuxt UI
- Tailwind CSS
